# career-test 

A simple site for DISC tests using nodejs + expressjs + reactjs on AWS (dynamo db)

## Running Locally

```sh
$ git clone ...
$ cd career-test
$ npm install
$ npm start
```
Your app should now be running on [localhost:5000](http://localhost:5000/).
